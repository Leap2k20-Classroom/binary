# Binary

## Даалгавар

`index.js` дотор байгаа `binaryCalc` функцыг дуусга.

binaryCalc функц нь параметраар 1 натурал тоо авах ба тухайн тооны 2 тын бичэглэл дэх тэгш дугаар дээр байгаа цифрүүдийн нийлбэр болох ганц тоо л буцаана 

Жишээ нь:
* `10` - ийг => `1010` =>  `2` гэж буцаана
* `8` - ийг => `1000` =>  `0` гэж буцаана

Зөвхөн **binaryCalc** гэсэн функцыг засахыг анхаарна уу!
